# Go-Live Checklist

## Hardware
- INPOS M530 connected to LAN with static IP
- CL3000 label scale configured with correct barcode format
- Receipt printer (ZY907) installed and named in Windows
- Label printer (X-Printer 490B) installed and named in Windows

## Software
- ERPNext site running and reachable on LAN
- POS Awesome installed and configured with POS Profile
- QZ Tray installed and running on cashier PC
- POS Printing Settings configured (`/app/pos_printer_setup`)
- Weighed barcode rules loaded and validated with sample barcodes

## Operations
- Cashiers trained on manual fiscal printing (MVP)
- Backup/export process defined (ERPNext)
- Offline procedure documented
